Vision PPO Racing Arena

Tune the reward. Train the policy. Race on a narrow sky road using camera frames only.

Vizuara student RL challenge - MetaDrive + PyTorch PPO

Students train a policy from raw RGB observations, not lidar and not simulator state. The leaderboard ranks checkpoints by route completion, then mean return, then shorter episode length. The goal is to design reward settings that make the car finish more of the track.

Challenge Setup

A compact reference for students before they start tuning rewards.

Observation

Camera onlyStacked RGB frames from MetaDrive.
Shape4 x 3 x 64 x 64 by default.
No shortcutsNo lidar and no state vector in the policy.

Action

Steering-1 left, +1 right.
ThrottlePositive accelerates, negative brakes.
PolicyCNN actor-critic with continuous Normal actions.

Reward

+
Forward progress
progress along sky road
+
Useful speed
moving forward without drifting
-
Crashes or falling
terminal penalty
-
Idle, brake, off-road
discourage stalling and exploits

How Students Submit

After training, submit a checkpoint from the same Python starter kit.

Train

python train_ppo.py --steps 100000 python train_ppo.py --steps 100000 --render

Submit

python submit.py --checkpoint checkpoints/policy.pt --tag my-agent-v1 --name "Student Name" --uid student-id

Optimize
